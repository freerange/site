Making WebGL Dance

I know hardly anything about computer graphics, have no real interest in WebGL, yet still found these slides from a presentation by Steven Wittens captivating. It’s amazing what can be accomplished in the browser these days, but more than just a demo they are informative too. I can’t wait for the talk video to become available. — TW

storytlr open source | your life online

I’ve just had a quick play with this indieweb life streaming application. The app itself is OK although I did have problems trying to import data from flickr and github.

The most interesting aspect was the “Direct Install” option that uses ComodIT to provision a new AWS instance and install the app for you. I wonder if I could use the recently announced Heroku Platform API to do something similar for Roosmarks or Money Tracker? — CR

Run a blog? Better fix your bounce rate

It turns out that Google Analytics will wait for a visitor to trigger a second event, such as visiting another page on the same site. If that second event is never triggered, the visit is counted as a bounce (regardless of the duration of the visit).

I found this really interesting as I suspect it’s a problem lots of people run into with a standard “install” of Google Analytics. — CR

Understanding Computation Book Club - Chapter 2

A few folk are gathering to read and discuss Tom’s book at GFRHQ. We’re going to be discussing Chapter Two, which is all about semantics: determining what a program actually means. You can read it for free here. You’re more than welcome to join us, although I’m starting to get worried that we’ll have enough chairs… — JA

Epic Rap Battles of History

Off the back of Youtube’s recent “Comedy Week”, I’ve been spending a lot of my non-work vegetative time gawping at videos on the internet. This is one series that I’ve really enjoyed; particular favourites include Mozart vs. Skrillex, Tesla vs. Edison and Einstein vs Hawking, but really, they’re all good. — JA

Flight, an event-driven framework from Twitter

While some web frameworks encourage developers to arrange their code around a prescribed model layer, Flight is organized around the existing DOM model with functionality mapped directly to DOM nodes.

I haven’t quite got my head round the pros and cons of this approach, but given that Dan Webb was involved in writing it, I’m sure it’s worth some serious attention. — JM

OAuth2 as a Hotel Token

I’ve played a lot with OAuth2 in the last few years, and while over recent months it feels like it has become mired in controversy and disagreement, there’s still much to like about the standard. This analogy from Tim Bray is the simplest I’ve seen to describe what an access token is. — TW

Ruby Object Mapper

There seems to be a surge in interest in Ruby object-relational mapping libraries. This one uses axiom a relational algebra library which I’d not come across before. I’ve also recently read a bit about datamappify which is built on top of existing ORMs like ActiveRecord and is based on the Repository Pattern and Entity Aggregation. — JM

Raildar

It’s great to see another website making use of Network Rail’s operational data feeds which were opened up last year. — JM
